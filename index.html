A Modded Adventure: Onward!

Onward! is a Fabric Minecraft Modpack for 1.18.x, featuring a mix of tech, magic, adventure etc, mods.

There will be no sacarfices on memory usage or performance, using mods like ferritecore and sodium.

The long term goal is a pack that's large and balanced, but also moving to the future, being managed with git and trying to be done in a rolling release.

The pack currently primarily experiments with alpha mods, but plans to shift over before a full release.
